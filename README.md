# likely

Shiny app inspired by Nicola Rennie's TidyTuesday submission shared at <https://bsky.app/profile/nrennie.bsky.social/post/3mgrh6qjnnb2w>.

Much of the code for the plot is inspired by the code provided at <https://github.com/rfordatascience/tidytuesday/blob/main/data/2026/2026-03-10/readme.md> for this submission.

This app is used in Lecture 18 of the course, in the "Interactive visualizations and reporting with Shiny" unit.

App is deployed at <https://minecr-likely.share.connect.posit.cloud/>.
